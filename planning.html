¿Qué es html5?

HTML5 (HyperText Markup Language, versión 5) es la quinta revisión del lenguaje HTML.
Esta nueva versión (aún en desarrollo), y en conjunto con CSS3, define los nuevos estándares
de desarrollo web, rediseñando el código para resolver problemas y actualizándolo así a nuevas
necesidades. No se limita solo a crear nuevas etiquetas o atributos, sino que incorpora muchas
características nuevas y proporciona una plataforma de desarrollo de complejas aplicaciones web
(mediante los APIs).

Algunas de las nuevas características de HTML5 serían:

  • Nuevas etiquetas semánticas para estructurar los documentos HTML, destinadas a remplazar la necesidad
    de tener una etiqueta
    que identifique cada bloque de la página.
  • Los nuevos elementos multimedia como audio y video
  • La integración de gráficos vectoriales escalables (SVG) en sustitución de los genéricos object, y un
    nuevo elemento canvas que nos permite dibujar en él.

La estructura y sintaxis en htmal5

En HTML5 destaca sobretodo la simplicidad y la permisividad.

  • EL DOCTYPE: La primera línea de un documento corresponde a la DTD (Document Type Declaration)
    o declaración de tipo de documento y sirve para indicar que versión de lenguaje HTML se ha usado al escribirlo.
  • EL DOCUMENTO HTML: En una página web, el elemento indica el inicio del contenido HTML. En HTML
    sólo necesitas indicar el lenguaje de la pagina.
  • LA CODIFICACIÓN DE LOS CARACTERES: Es frecuente indicar qué codificación de caracteres se ha usado
    en una página web. Con HTML5 se simplifica con lo indispensable para que los navegadores puedan gestionarla
    correctamente.
  • LOS SCRIPTS: estan escritos en Javascript.

LA SINTAXIS DE LOS ELEMENTOS

Las comillas en HTML5 pasan a ser facultativas para los valores de los atributos. Puedes escribirlos con
comillas dobles, comillas simples o sin ellas. Algunas pautas a tener en cuenta son:

  • Usa sólo minúsculas
  • Usa siempre comillas para los valores de los atributos
  • Cierra con / las etiquetas con una sola etiqueta de apertura

Etiquetas mas utilizadas en html5

  • Etiqueta Title: Define un título para el documento
  • Etiqueta meta: Define un metadato de un documento
  • Etiqueta a: Define un hipervínculo
  • Etiqueta br: Define un salto de línea
  • Etiqueta h1 a h6: Define encabezados o títulos
  • Etiqueta img: Define una imagen
  • Etiqueta p: Define un párrafo
  • Etiqueta style: Define un estilo para la información de un documento
  • Etiqueta table: Define una tabla
  • Etiqueta video: Define un vídeo o película
  • Etiqueta b: Define texto en negrita
